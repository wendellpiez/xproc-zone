XProc 3.0/3.1 Step List - easy lookup, with links.

This listing of XProc steps is designed to be used in conjuction with other resources, in particular the pages it links to (in specification documents and in Erik Siegel's XProcRef site).

The step list is not a comprehensive reference to XProc elements - while all of XProc except the steps is described in the Core Specification especially the section Other pipeline elements. These elements include p:variable, p:input, p:output, p:option, p:documentation and others.

If you find errors or lapses - or you have other good ideas - please let them be known.

Generated with the help of XProc, an Index to XProc shows XProc element usage in XProc pipelines throughout this repository. The polling is dynamic and can be executed at any time, including under continuous deployment and integration (CI/CD).

Look up XProc elements of interest and link from their listings into the code base.
