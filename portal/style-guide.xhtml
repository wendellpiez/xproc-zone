Panel One

Panel one goes here, Linking and with emphasis as well.

The styling supports two main kinds of div displays in addition to the default plus whatever you might add. A panel is what you see here. An example of a block is the details (expanding box) above, or the section entitled Block below. Any HTML element can be a panel or a block, marked as such using its class attribute in the normal way.

Panel Two

Panel two goes here.

The emphasis color can be gotten by using class="hi" or highlight on anything at all. We can extend this vocabulary as needed for more effects.

By applying a class value hanging, it is possible to achieve a hanging indentation for a paragraph, useful (for example) for lists of bibliographic citations.

Block

Not marked as a panel, but marked as a block. Panels are more distinctive.

But not everything has to be in a panel or block.
